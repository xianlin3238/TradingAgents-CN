## 📦 技术架构

- **后端**：FastAPI + Uvicorn
- **前端**：Vue 3 + Vite + Element Plus
- **数据库**：MongoDB + Redis
- **部署**：Docker 多架构支持 (amd64 + arm64)

## 🎯 核心功能

- 多LLM提供商集成 (OpenAI, Gemini, DeepSeek, 通义千问等)
- 智能模型选择
- 用户权限管理
- 批量分析功能
- 模拟交易系统
- 专业报告导出 (Markdown/Word/PDF)
- 完整A股支持
- 中文本地化
